# IPL Data Analysis 🏏

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on IPL (Indian Premier League) dataset to uncover insights about teams, players, and match outcomes.

---

## 📂 Dataset

* matches.csv
* deliveries.csv *(not included due to file size limit)*

---

## 🛠️ Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## ⚙️ Workflow

1. Data Cleaning
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Visualization

---

## 📊 Key Analysis

* Team performance over seasons
* Player statistics (runs, wickets, etc.)
* Toss impact on match results
* Venue-based performance

---

## 📈 Key Insights

* Teams winning toss often prefer fielding
* Certain teams dominate specific venues
* Top players consistently impact match outcomes

---

* Python
* Jupyter Notebook
